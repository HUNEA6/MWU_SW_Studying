HTML과 CSS
HTML(Hyper Text Markup Language)
- 구조(Structure)와 의미(Semantic)로만 남게되었다.

HTML5
- 2014년 10월 28일에 표준안이 확정되었다.
- 영상과 음성을 처리하기 위한 멀티미디어 요소가 추가되었다.
- 구조적인 요소가 추가되었다.
- 표현을 담당하던 태그가 삭제되고 의미를 담은 태그가 추가되었다.

* 태그마다 기본 스타일이 있다. (ex: li, ul)
태그마다 기본 스타일이 있는 것이 마음에 들지 않는다면
https://meyerweb.com/eric/tools/css/reset/
모든 HTML 속성을 제거해준다.

* 브라우저마다 미묘하게 기본 스타일이 다르다.
마음에 들지 않는다면
http://necolas.github.io/normalize.css/
브라우저간 차이를 통일시켜 준다.
