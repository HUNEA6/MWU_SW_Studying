정규표현식
- 패턴을 이용하여 문자열에서 원하는 문자를 검색(search), 대체(replace), 추출(extract)할 수 있다. 
- 매우 편리한 기능이나 성능이 매우 느리다.

JavaScript에서 사용하는 방법
생성 방법
- JavaScript는 RegExp 객체로 정규표현식 기능을 제공한다.
- Array, Object처럼 Literal로 생성 가능하다.

정규 표현식 객체
1. test 함수
- 정규표현식 객체의 test함수는 입력받은 문자열에 찾는 패턴이 있는지 찾은 후 있다면 true, 없다면 false를 반환한다.
2. exec 함수
- 정규표현식 객체의 exec 함수는 입력받은 문자열에 찾는 패턴이 있는지 찾은 후 일치한 패턴 정보를 반환하고 없으면 null을 반환한다.
- 문자 추출에 해당한다.
3. match 함수
- String 객체의 match 함수는 정규표현식 객체를 파라미터로 받아 패턴이 있는지 찾은 후 일치한 패턴 정보를 반환하고 없으면 null을 반환한다. 정규표현식 객체의 exec 함수와 같다.
- 문자 추출에 해당한다.
4. replace 함수(★ 상당히 자주 쓰임)
- String 객체의 replace 함수는 정규표현식 객체를 파라미터로 받아 패턴이 있는지 찾은 후 일치한 패턴 정보를 원하는 문자열로 바꿀 수 있다. 
- 문자 대체에 해당한다.
5. search 함수
- String 객체의 search 함수는 정규표현식 객체를 파라미터로 받아 패턴이 있는지 찾은 후 일치한 패턴 정보의 위치를 반환한다.
- 문자 검색에 해당한다.
6. capture 함수
- 캡처가 적용된 정규표현식을 이용하면 match 반환값의 1번 인덱스부터 순차적으로 캡처 결과가 들어간다.
