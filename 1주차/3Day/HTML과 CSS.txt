이후에 알아볼 것
- CSS에서 id와 class의 차이점은?
ID는 하나의 태그가 개별적인 스타일을 가질 수 있는 반면,
CLASS는 여러 태그가 종합적인 스타일을 가질 수 있는 차이가 있다.

- 만약 id가 여러 개 있다면 어떻게 처리될까?
하나의 태그에 ID 속성은 하나만 가질 수 있고 CLASS 속성은 여러 개를 가질 수 있다.
CLASS 속성을 둘 이상 사용할 때는 class="class1 class2" 와 같이 띄어쓰기로 구분해야 한다.
ID 또는 CLASS를 여러 개를 가질 때 스타일이 중복되지 않도록 해야 한다.
하나의 태그가 여러 개의 클래스 속성을 가진다면 
스타일 코드가 중복되지 않도록 클래스를 잘 분류하는 것이 중요하다.
